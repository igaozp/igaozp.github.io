使用链表实现 Bag 背包类

Bag 背包类可以用于元素的收集和遍历，类似于 Set 集合，但不可以删除其中的元素。在 Bag 内部使用节点类 Node 来作为链表的每一个节点元素，同时使用链表基本的头插法来实现元素的插入，属于链表最基本、最基础的应用。为了实现元素的迭代， Bag 通过自定义的内部类 ListIterator 来实现 Iterable 接口。
